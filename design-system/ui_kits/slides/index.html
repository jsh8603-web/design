SECTION 3

Action titles, not topic labels.

PF · the cardinal rule
❌ TOPIC LABEL
Q3 매출
A topic label tells the audience what the slide is "about". It hides the conclusion.
✓ ACTION TITLE
Q3 매출이 전년 대비 23% 증가했다
An action title is an assertion. The audience can extract the takeaway in 1 second.
SECTION 4

PF rules

pf_rules.md · enforced by preflight
PF-66 · NO box-shadow
PPTX strips all shadows. Convey elevation via surface tint.
PF-63 · NO <table>
Use CSS grid with explicit grid-template-columns instead.
PF-42 · NO opacity < 1.0
Backgrounds must be solid hex. Blend by hand if needed.
PF-25 · MIN 10pt
24/14/10 floor for title/body/caption. Never go smaller.
PF-26 · MAX 3 blocks
One slide = one message. Split into more slides instead.
PF-28 · ≤80 CJK words
120 EN / 80 CJK ceiling. Captions and sources don't count.
